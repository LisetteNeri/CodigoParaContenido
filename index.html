¿Qué es el CSS?

Cascading Style Sheets / Hojas de Estilo en Cascada. Describe la presentación de documentos escritos en un lenguaje de marcado como HTML, sirve para dar estilos y una representación visual atractiva para el usuario

¿Qué es una Regla CSS?

Una regla se compone de un **Selector**  qué es el elemento al que afecta y un bloque de **Declaraciones** el estilo que se le aplica.
